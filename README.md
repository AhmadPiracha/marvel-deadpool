# DEADPOOL

## 📋 Prerequisites

- Node.js (v14.0+)
- npm or yarn

## ⚙️ Installation

```bash
# Clone the repository
git clone https://github.com/ahmadpiracha/marvel-deadpool.git

# Navigate to the project directory
cd marvel-deadpool

# Install dependencies
npm install

# Start the development server
npm run dev
```

## 🛠️ Built With

- **Frontend**: React.js, TailwindCSS, GSap
